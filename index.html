Website Development: ubutheatre.com

Date: 2012 - Present Role: Lead Developer Language: PHP

This is perhaps my most public example of software development. The website is a custom-written PHP application, which processes ticket bookings for theatrical performances at the University of Bristol Students' Union. My involvement began after the site was a few years old, and involved re-working a lot of the code base to conform to more modern standards of security, object orientation and database normalisation. I introduced source control to the project, along with issue tracking and a system for managing support requests.

I am also responsible for the visual design (pictured). The live site can be found here, although it is still no where near the stage where it could be called complete.

Summer Internship at NATS

Date: Summer 2012 Role: Summer Placement Student Language: C++

In the summer of 2012, I completed a 3 month internship at National Air Traffic Services, the company that provides air traffic control to the majority of the UK. My task was to specify, design and build a component of their simulation software. This component simulated the operational communications system in use between controllers, and was built using the Qt toolkit and written in C++.

Image Processing: Gesture Recognition

Date: Autumn 2013 Language: C++

Working in a group of 3, we produced an implementation of the Lucas and Kanade algorithm to estimate the motion field over a set of frames. Using the output from this, we made a decision as to whether the motion was predominantly left or right, and outputted this to the display window. We used the OpenCV image processing library.
