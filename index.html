Website Development: ubutheatre.com

Date: 2012 - Present Role: Lead Developer Language: PHP

This is perhaps my most public example of software development. The website is a custom-written PHP application, which processes ticket bookings for theatrical performances at the University of Bristol Students' Union. My involment began after the site was a few years old, and involved re-working a lot of the codebase to conform to more modern standards of security, object orientation and database normalisation. I introduced source control to the project, along with issue tracking and a system for managing support requests.

I am also responsible for the visual design (pictured). The live site can be found here, although it is still no where near the stage where it could be called complete.

Image Processing: Gesture Recognition

Date: Autumn 2013 Language: C++

Working in a group of 3, we produced an implementation of the Lucas and Kanade algorithm to estimate the motion field over a set of frames. Using the output from this, we made a decision as to wether the motion was predomenantly left or right, and outputed this to the display window. We used the OpenCV library.

Machine Learning: Spam Filter

Date: Autumn 2013 Language: Java

This was a pair project, where we had to implement a spam filter using the Naive Bayes method. We split the program into 3 classes: the main filter program which was given a filename and had to output spam or ham; the NaiveBayesClassifier class, which did the actual classificaiton, and the TrainClassifier class, which produced and saved a trained version of the NaiveBayesClassifier.
